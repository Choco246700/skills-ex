Project Presentation

Skills EX

A community-driven skill exchange platform designed to bridge knowledge gaps through reciprocal learning and mentorship.
Developed by: Group 19
Supervisor: Project Faculty Advisor
The Vision

Solving Knowledge Silos

Most student communities struggle with:
  • Information Asymmetry: Experts don't know who needs help.
  • Barriers to Entry: High costs for specialized tutoring.
  • Passive Learning: Lack of practical, peer-to-peer exchange.
Innovation

Core Feature Set

⚡ Live Pulse Search

Real-time skill discovery with instant suggestions as you type.

🎉 Match Celebration

Visual delight system using confetti algorithms to reward connection.

🎖️ Dynamic Milestones

Automated badge system (Verified, Expert) for community trust.

🌗 Pro Night Mode

Accessibility-first dark mode with persistent state management.

Administration

Admin Category Pulse

Managing a community requires data. Our Admin Portal provides:
  • Live Distribution: Real-time visualization of skill categories.
  • Operational Insight: Quick stats on connections and active members.
  • Community Moderation: Full registry management for safe exchange.
